# Module 2: Basic Setup & AI Model Selection

After installation, OpenClaw requires some fundamental settings to be configured. The CLI Setup Wizard will guide you.

### 2.1 File Permissions
Once OpenClaw starts, it will request read and write permissions to your computer layout and system.
- Provide the requested access to allow OpenClaw to handle background tasks effectively.

### 2.2 Fresh Configuration
The setup wizard simplifies configuration. If you've tinkered with previous installations, completely reset and configure the assistant step-by-step alongside the wizard.

### 2.3 Choosing the AI Model
OpenClaw supports multiple Large Language Models to power its interactions.
You will be prompted to select an AI Model logic. For a cost-free and accessible option:
- Select the **ChatGPT Codex** model.

**Note:** This model uses an OAuth login, providing free weekly credits for use—making it ideal to test out features without inputting payment details.
