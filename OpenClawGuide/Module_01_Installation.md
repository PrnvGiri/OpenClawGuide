# Module 1: Downloading & Installing OpenClaw

## Prerequisites
- **Node.js**: Ensure it is installed.
- **Git**: Ensure it is installed.

## Installation Methods

### Method 1: Installing via NPM (For existing Node.js Users)

If you already have Node.js and Git installed, use the NPM global installation command:

```bash
npm i -g openclaw
```
Once installed, start the configuration wizard:
```bash
openclaw onboard
```

### Method 2: The Easiest Windows Installation Script (PowerShell)

For a one-click installation on a Windows Machine, simply copy and run the following command directly into a terminal or PowerShell instance:

```powershell
iwr -useb [https://openclaw.ai/install.ps1](https://openclaw.ai/install.ps1) | iex
```
